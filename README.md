# Flashcard app

## Description

### This website provides a useful tool to create/delete study decks. It presents the deck in a carousel view. 

Check out [this project](https://sc429.github.io/ai-se_project_flashcards/)

## Feature

### This app provides multiple features of creating your own deck and adding content to help you study. Once you select a deck, it will switch the view to a carousel where you can switch between cards and flip them between questions and answers.

## Technologies used

### The app mainly uses a combination of HTML and CSS for frontend and Javascript for backend.
